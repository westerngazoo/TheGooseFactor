# Unit I: Vectors & Spaces

Linear algebra begins with the **vector space**: a set of objects you can add and scale. In Geometric Algebra these grade-1 objects live inside a larger algebra, and the **outer product** $\wedge$ gives us an immediate, computational test for the most important property of a collection of vectors — linear independence. A set is independent precisely when its wedge is nonzero.
