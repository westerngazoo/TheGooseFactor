# Poems

Welcome to the poems section. Add new poems as separate Markdown files under `books/poems/`.

## Tips
- Create one file per poem (e.g., `my-poem.md`).
- Add frontmatter with `sidebar_position` and `title` to control ordering.
- Push to the repo and the poem auto-appears in the sidebar.
