# Rust: Algorithms & Data Structures

> A guide to classical algorithms and data structures, written in Rust.

## Why Rust?

Rust offers memory safety without a garbage collector. Learning algorithms in Rust helps you understand ownership, borrowing, and lifetimes while mastering fundamental computer science concepts.
