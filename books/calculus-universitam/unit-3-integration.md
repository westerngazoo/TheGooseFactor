# Unit III: Directed Integration

Integration is the summation of quantities, but in Geometric Algebra the *measure itself carries orientation*. Instead of a bare scalar $dx$, we integrate against a directed element $d\mathbf{x}$ (a vector), $d\mathbf{A}$ (a bivector area), or $d\mathbf{V}$ (a trivector volume). This single shift makes the fundamental theorem of calculus a statement about boundaries that will generalize, in Unit V, into the one theorem that contains Green, Gauss, and Stokes.
