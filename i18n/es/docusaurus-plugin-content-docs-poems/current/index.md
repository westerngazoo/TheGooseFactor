# Poemas

Bienvenido a la sección de poemas. Añade nuevos poemas como archivos Markdown separados bajo `books/poems/`.

## Consejos
- Crea un archivo por poema (ej., `mi-poema.md`).
- Añade frontmatter con `sidebar_position` y `title` para controlar el orden.
- Empuja al repo y el poema aparece automáticamente en la barra lateral.
