# 每个人都应该了解的两个训练传统

大多数训练者只学一个传统 — 第一个教练用的那个 — 然后就停了。这是个错误。有两种方法尤其产生了不成比例的有用证据,而且不在大多数健身房的常规课表里:

1. **肌腱韧性训练** — KOT(Knees Over Toes)/ Ben Patrick 学派 + 经典的大重量慢速抗阻(HSR)肌腱康复研究。
2. **苏联共轭法 / Westside / Verkhoshansky** — 过去 40 年最强力量举运动员背后的方法,源头是 1960–80 年代的苏联运动科学。

两者都值得在[动作库](/apps/exercise-library)里占据一席之地,也值得放进你的每周计划。这是导览。

## I — 肌腱韧性:Ben Patrick 学派

### 前提

标准的力量课表训练肌肉。**它几乎不训练肌腱。** 肌腱重塑很慢(数月,不是数周),对特定刺激有反应,并且是 35 岁以上大多数训练者的限制因素。

如果你曾经:
- 深蹲时听到髌骨"啪"的一声
- 冲刺时感觉跟腱"嗖"的一下
- 患过髌腱病("跳跃膝")
- 患过跟腱病
- 跑步时膝盖内侧疼痛

— 你已经见识过后果了。大多数训练者花了多年时间堆股四头肌的肌肉量,却从未用髌腱需要的完整范围去加载它,以让它有韧性。然后一场临时打的篮球游戏就结束了举铁生涯。

> :sharpgoose: 肌肉以周为单位响应负荷。肌腱以月为单位响应。如果你不专门训练肌腱,你就有一个力量失衡,迟早会找到你。

### Ben Patrick(Knees Over Toes Guy)

Ben Patrick 的 KOT 方法在 2019 年爆火 — 病毒式传播的篮球准备内容。原则:

1. **以完整范围训练膝关节**,包括"膝过脚尖"。1980 年代健身房里"膝盖永远不该超过脚尖"的说法是错的;肌腱需要那个范围。
2. **反向雪橇拖**。向后拖雪橇是对股四头肌肌腱最纯粹的刺激,**零离心负荷** — 适合伤后重建,也适合作为每日 GPP 工具。
3. **胫骨前肌提升**。训练不足的小腿前侧肌肉。KOT 让它进入主流;结果是更少的胫前疼痛和更好的膝关节轨迹。
4. **斜板训练**。脚跟在 25° 的斜板上;做 ATG 深蹲。最大化 VMO 募集。
5. **Patrick step**。从一个垫高处做慢速离心台阶下降。KOT 的招牌动作。

### 经典 HSR — 研究验证的同伴

早在 KOT 火起来之前,Per Kongsgaard 和 Stig Magnusson(哥本哈根)就做了一系列研究,确立了**大重量慢速抗阻(HSR)**作为肌腱康复的金标准:

- 每次动作向心 3 秒,离心 3 秒
- 每组 6 次,4 组,逐步加到约 85% 1RM
- 每周 3 次训练,最少 12 周

肌腱在这种精确的加载模式下重新组织其胶原结构。没有弹跳,没有跳跃训练,直到肌腱重塑完成。

Alfredson 方案(离心提踵治疗跟腱病)是原型 — 慢速离心,每天,持续 12+ 周。

### 肌腱经典(现在在库中)

这些动作在 Goose Factor 数据库里,笔记中带 KOT/HSR 标签:

| 动作 | 目标 | 笔记 |
|---|---|---|
| **ATG 分腿蹲** | 髌腱,VMO | 前脚抬高,膝过脚尖深度 |
| **Patrick Step** | 髌腱 | 从垫高处做慢速离心台阶下降 |
| **斜板深蹲** | VMO,膝 | 脚跟在 25° 斜板上,深 ROM |
| **Spanish Squat** | 髌腱 | 弹力带绕膝,带等长味道 |
| **胫骨前肌提升** | 胫骨前肌 | 背对墙站立,抬起脚尖 |
| **负重胫骨前肌提升** | 胫骨前肌 | 杠铃片或壶铃放在脚尖上 |
| **反向雪橇拖** | 股四头肌,VMO,胫骨前肌 | 向后走,无离心 |
| **哥本哈根平板** | 内收肌 | 侧平板,上腿放长凳 |
| **大重量慢速抗阻提踵** | 跟腱 | 3 秒上 / 3 秒下,6-15 次 |
| **单腿提踵(Alfredson)** | 跟腱 | 慢离心,康复金标准 |
| **反向北欧弯举** | 股四头肌 / 髋屈肌 / 髌骨 | 跪姿向后倾,控制下降 |
| **KOT Hip Airplane** | 臀中肌 | 单腿平衡配髋部旋转 |

> :weightliftinggoose: 两组 ATG 分腿蹲 + 一组 Patrick step + 一组反向雪橇拖 = 20 分钟的肌腱训练。每周塞两次。你的膝盖在 6-8 周后不再疼。

### 计划

肌腱训练是**预防/维护,不是主要动作**。典型的放置:

- **热身**(5-10 分钟):胫骨前肌提升、徒手 ATG 分腿蹲、Spanish squat。
- **辅助动作**(力量训练后):2-3 组 Patrick step 或斜板深蹲。
- **休息日 GPP**:中等负荷的反向雪橇拖 20 分钟。

每周两到三次就足够了。数月的一致性比任何单次训练的强度更重要。

## II — 苏联共轭法:Verkhoshansky 与 Simmons 学派

### 前提

线性周期化(累积到 1RM 峰值,然后重置)在西方做了几十年的默认方案。1960–80 年代与举重和田径运动员合作的苏联运动科学家产出了一个更好的系统:**同时训练多种素质,以轮换的焦点,配合适应性阻力**。

Westside Barbell 的 Louie Simmons 在 80 年代将其改编用于力量举。Yuri Verkhoshansky 在《Supertraining》(与 Mel Siff 合著)和《Special Strength Training Manual for Coaches》中发布了经典的苏联方法。Pavel Tsatsouline 的 RKC 引入了同一传统的壶铃一面。

> :angrygoose: 1985 年"西方"训练者还在跑 5×5 的进阶,抱怨平台期,而苏联人已经发布了一整套穿越它们的系统。信息就在那里 — 翻译滞后了 30 年。

### Westside 四日模板

经典的一周:

| 日 | 焦点 | 关键概念 |
|---|---|---|
| **一** | 大重量(ME)下肢 | 做到 1-3 RM 的某个轮换动作(box squat、deficit DL、banded GM)|
| **三** | 大重量(ME)上肢 | 1-3 RM 的某个轮换推举(floor press、pin press、board press)|
| **五** | 速度(DE)下肢 | 8-12×2 速度深蹲配弹力带,杠重 50-60% |
| **日** | 速度(DE)上肢 | 8-9×3 速度卧推,杠重 50-60% + 弹力带 |

每次训练后还有辅助动作。辅助动作不按周变化;主要动作每 1-3 周轮换一次,避免神经陈旧。

### 苏联 / Westside 经典

| 动作 | 概念 | 苏联 / Westside 血脉 |
|---|---|---|
| **速度卧推(DE)** | 速度训练 | 每一次都是补偿性加速 |
| **速度深蹲(DE)** | 速度训练 | Box squat + 弹力带,50-60% |
| **速度硬拉(DE)** | 速度训练 | 从地板尽可能快地拉起 |
| **带弹力带的箱式深蹲** | 适应性阻力 | 顶峰时通过弹力带变重 |
| **带弹力带的卧推** | 适应性阻力 | 同上,上肢 |
| **带弹力带的早安式** | ME 后链 | Westside 下肢辅助动作 |
| **反向超伸** | 脊柱减压 + GHR | Louie Simmons 的标志性发明 |
| **腰带深蹲** | 大重量股四头,无脊柱负荷 | Sorinex / Westside |
| **箱式深蹲** | 坐回去,爆发上升 | Westside 主日主力 |
| **臀腿挺身** | 后链孤立 | 苏联起源 |
| **跳深** | 反应力量 | Verkhoshansky 的冲击方法 |
| **高度下落** | 反应准备 | 稳稳着陆,不反弹 |
| **集组深蹲** | 85%+ 的容量 | 苏联强度-密度 |
| **波浪加载(3-2-1)** | 神经系统增强 | 两个上升波 |
| **钟摆抓举** | 反应性奥举 | Bondarchuk 原则 |
| **RKC 平板** | 张力训练 | Pavel Tsatsouline / RKC |
| **长循环挺举(壶铃)** | 体能 + 力量 | RKC 竞赛标准 |

### 需要内化的核心原则

**1. 补偿性加速。** 每一次都尽可能快地完成,无论负荷如何。带最大意图移动的 90% 深蹲,招募的快肌纤维与以同样方式移动的 50% 深蹲相同。速度力量会转移到最大力量。

**2. 适应性阻力。** 弹力带和铁链在范围内增加负荷 — 最重的地方在你最强的位置(顶峰),最轻的地方在你最弱的位置(底部)。训练你穿过完整 ROM 加速。

**3. 共轭轮换。** 主动作每 1-3 周变(floor press → board press → pin press → 反带卧推 → 回到 floor press)。辅助动作保持不变。这避开了西方线性计划里"频率 × 强度 = 陈旧"的陷阱。

**4. ME(大重量努力)≠ rep maxes。** 箱式深蹲的 1RM 与自由深蹲的 1RM 不是同一个动作。轮换 ME 变种让你能每周做大重量,而不会在*同一个*特定动作上烧光。

**5. 速度(DE)是必修。** 大多数西方训练者跳过它。Westside 训练者从不跳过。速度训练不是可选 — 这就是你保持爆发力的方式。

> :surprisedgoose: Westside Barbell 一个健身房产出的 1000 磅深蹲数量,比整个美国其他地方两个十年加起来还多。这套方法*管用*。美国力量举圈把它无视了,因为 Louie Simmons 性格扎人,而这个系统"看起来"不像传统训练。

### 计划

如果想试试苏联共轭风格,而不全套上 Westside,最便宜的入门是:

- **周一**:在你现有的主动作上做大重量(1×5 或 3×3)
- **周三**:轻 / 速度日 — 6×3 在 60% 配爆发意图
- **周五**:辅助 + 反向超伸 + GHR + 胫骨前肌

这就是穷人版的共轭周,而且管用。完整系统需要弹力带、铁链、反向超伸机,以及一个懂这个程序的教练。大多数商业健身房没有这些设备。

## III — 这些动作在 Goose Factor 工具箱里的位置

动作库现在有 **300+ 个动作**,包括为这两个传统刚加入的 ~22 个。在动作库里按名称搜索任何一个 — 每条记录的笔记字段都引用了方法学。

在[训练计划生成器](/apps/routine-generator)和[训练构建器](/apps/session-builder)里,肌腱训练倾向于落在热身 / 泵感阶段;苏联 ME 动作落在大重量阶段;DE 动作也落在大重量阶段,但带速度偏向标注。

如果你在构建个人的每周模板:

- **两天的肌腱训练**作为热身或休息日 GPP。反向雪橇拖是膝盖耐用性的最高杠杆单项动作。
- **每个主动作一天 Westside 风格的 ME** — 每 2-3 周轮换变种。
- **每个主动作一天 DE** 训练速度力量。

大多数把两者结合起来的训练者报告:更少的关节问题、8-12 周内更好的 1RM,以及到第 3-4 个月时显著的韧性提升。
