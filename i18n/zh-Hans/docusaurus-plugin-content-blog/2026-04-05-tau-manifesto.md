# 欧拉公式很美 — 但只有代入 Tau,不是 Pi

定义 $\tau = 2\pi \approx 6.283185\ldots$

就这样。一圈。单位圆的周长。正弦和余弦的周期。300 年来在数学、物理和工程的每一个公式里,$2\pi$ 笨拙地代替的那个东西。

这不是审美偏好。这是技术论证。每一个包含 $2\pi$ 的公式都是我们选错了常数的证据。而这是很多公式。

<!-- truncate -->

> :angrygoose: 先把这件事说清楚:我不是来辩论的。$\pi$ 错了。不是这个数 — 是这个*选择*。如果你还在到处写 $2\pi$ 并想着"没事啊,这只是约定俗成,"你已经被 300 年的数学传统 gaslight 了。系好安全带。

## 核心问题

圆周率应该是周长对**半径**的比,而不是周长对直径的比:

```math
\tau = \frac{C}{r} = 6.283185\ldots
```

我们到处都在用半径。半径定义单位圆。半径参数化极坐标。半径出现在每个距离公式、每个范数、每个度量空间里。直径是导出量 — 两个半径粘在一起。然而 $\pi = C/d$ 是按直径定义的。

这意味着每次圆周率出现在它的自然语境里,我们写 $2\pi$ 而不是 $\tau$。那个因子 2 是为历史事故记账。它不增加任何信息。它就坐在那里,堵塞公式,制造符号错误的机会。

> :nerdygoose: 历史上,有些早期数学家*确实*用过类似 $\tau$ 的常数。William Jones 在 1706 年引入了 $\pi$,但欧拉在 1736 年把它推广开 — 他实际上在不同的论文里用 $\pi$ 表示不同的东西,有时是 $3.14...$,有时是 $6.28...$。在半圈常数上的标准化基本上是任意的。我们从那时起就一直在承担后果。

## 角度:最明显的地方

绕圆一整圈是 $\tau$ 弧度。半圈是 $\tau/2$。四分之一圈是 $\tau/4$。

| 圈的分数 | 用 $\tau$ | 用 $\pi$ |
|---|---|---|
| 一整圈 | $\tau$ | $2\pi$ |
| 半圈 | $\tau/2$ | $\pi$ |
| 三分之一圈 | $\tau/3$ | $2\pi/3$ |
| 四分之一圈 | $\tau/4$ | $\pi/2$ |
| 六分之一圈 | $\tau/6$ | $\pi/3$ |
| 八分之一圈 | $\tau/8$ | $\pi/4$ |
| 十二分之一圈 | $\tau/12$ | $\pi/6$ |

用 $\tau$,一圈的分数字面上就是分母。四分之一圈就是 $\tau/4$。十二分之一圈就是 $\tau/12$。记号直接映射到几何。

用 $\pi$,四分之一圈是 $\pi/2$ — 看起来像"某物的一半"。一半的什么?一半的 $\pi$,也就是半圈。所以四分之一圈是半圈的一半。两层间接才能表达最简单的几何概念。

> :sarcasticgoose: "可我已经用 $\pi$ 记住单位圆了!" 好。现在向学生解释为什么四分之一圈是 $\pi/2$,而且不能在一句话里用两次"一半"。我等你。

学生花数周建立弧度制的直觉。其中大部分时间是在和因子 2 搏斗,而不是学习几何。

## 单位圆的值

用 tau,标准角变得透明:

| 角度 | $\tau$ 形式 | $\sin$ | $\cos$ |
|---|---|---|---|
| $0$ | $0$ | $0$ | $1$ |
| $\tau/12$ | 30 度 | $1/2$ | $\sqrt{3}/2$ |
| $\tau/8$ | 45 度 | $\sqrt{2}/2$ | $\sqrt{2}/2$ |
| $\tau/6$ | 60 度 | $\sqrt{3}/2$ | $1/2$ |
| $\tau/4$ | 90 度 | $1$ | $0$ |
| $\tau/2$ | 180 度 | $0$ | $-1$ |
| $\tau$ | 360 度 | $0$ | $1$ |

到 $\tau/4$ 你走了四分之一圈。到 $\tau/2$ 你走了一半。到 $\tau$ 你回到原点。角度就是圈的分数,直接。无需心算转换。

> :mathgoose: 这是用 $\tau$ 的助记。$0, \tau/12, \tau/8, \tau/6, \tau/4$ 处的正弦值是 $\sqrt{0}/2, \sqrt{1}/2, \sqrt{2}/2, \sqrt{3}/2, \sqrt{4}/2$。也就是 $0, 1/2, \sqrt{2}/2, \sqrt{3}/2, 1$。余弦是同样的序列倒过来。干净。现在试着从 $\pi$ 的记号里看出这个模式。你看不出来,因为分母在骗你。

## 周期性

正弦和余弦的周期是 $\tau$:

```math
\sin(\theta + \tau) = \sin\theta, \qquad \cos(\theta + \tau) = \cos\theta
```

基本圆函数的周期就是圆周率。本该如此。写 $\sin(\theta + 2\pi) = \sin\theta$ 让周期看起来像导出量,模糊了这一点。

## 欧拉恒等式 — 真正的那个

"数学中最美的方程"通常写作:

```math
e^{i\pi} + 1 = 0
```

> :surprisedgoose: 快 — 这个方程在几何上*意味着*什么?"$e$ 的 $i$ 乘以半圈次方等于负一。" 好,在复平面里绕单位圆走一半到 $-1$。这是... 一个特殊情况。一个挺特殊的特殊情况,但还是特殊情况。

更深的恒等式是欧拉公式应用到一整圈:

```math
e^{i\tau} = 1
```

把 $e$ 提到复平面里一整圈的次方,把你带回起点。*这* 才是基本陈述。恒等式 $e^{i\pi} = -1$ 是推论,不是反过来。

> :angrygoose: $\pi$ 版本被称为"美",因为它在一个方程里有 $e$、$i$、$\pi$、$1$ 和 $0$。酷,所以我们现在在为拼字游戏分数做优化?$\tau$ 版本,$e^{i\tau} = 1$,*在说事情*:完整旋转就是恒等变换。这不只是漂亮 — 这就是复平面里"一整圈"的定义。美是真理,不是收集符号。

$\tau$ 形式也能干净地推广。$n$ 次单位根是:

```math
e^{i k\tau/n}, \qquad k = 0, 1, \ldots, n-1
```

单位圆上均匀分布的点,由 $\tau$ 的分数参数化。用 $\pi$,你写 $e^{2\pi i k/n}$ — 因子 2 又回来了,什么都没加。

## 周长与面积

**周长**:$C = \tau r$。干净。一个常数,一个半径。

用 $\pi$:$C = 2\pi r$。因子 2 是代数噪音。

**面积**:$A = \frac{1}{2}\tau r^2$。

> :sarcasticgoose: "啊哈!$1/2$ 丑了!$\pi r^2$ 更干净!" 我就知道你会这么说。这是 $\pi$ 支持者唯一的论据,而且它错了。继续读。

看结构。$A = \frac{1}{2}\tau r^2$ 与下面这些有完全相同的形式:

```math
\frac{1}{2}mv^2 \quad\text{(动能)}, \qquad \frac{1}{2}kx^2 \quad\text{(弹簧能)}, \qquad \frac{1}{2}at^2 \quad\text{(匀加速距离)}
```

这些都来自对线性函数积分。圆的面积是 $\int_0^r \tau\rho\,d\rho = \frac{1}{2}\tau r^2$ — 半径 $\rho$ 处的周长,从 0 到 $r$ 累积的积分。$1/2$ 不尴尬 — 它是由积分产生的二次函数的标志。用 $\pi$ 把它隐藏,模糊了微积分。

> :mathgoose: 这是杀手论据。$\frac{1}{2}\tau r^2$ 里的 $1/2$ 告诉你*公式从哪里来* — 它是线性函数的反导数。每个物理学生都知道 $\frac{1}{2}mv^2$ 是动能,来自对动量的积分。同样的结构。$\pi r^2$ 记住了答案但毁掉了推导。它相当于代码里的魔法数字。

## 高斯分布

正态分布:

```math
f(x) = \frac{1}{\sqrt{\tau}\,\sigma}\,e^{-\frac{(x-\mu)^2}{2\sigma^2}}
```

用 $\pi$:$\frac{1}{\sqrt{2\pi}\,\sigma}$。$\sqrt{2\pi}$ 其实是 $\sqrt{\tau}$。每本统计教科书都把 $\sqrt{2\pi}$ 当成不透明的归一化常数扛着。用 $\tau$,它变透明:高斯函数的归一化因子是 $\sqrt{\tau}$。

> :nerdygoose: 如果你曾经好奇为什么正态分布里有个 $\sqrt{2\pi}$ — 是因为高斯积分 $\int_{-\infty}^{\infty} e^{-x^2/2}\,dx = \sqrt{2\pi} = \sqrt{\tau}$。$2\pi$ 不是两个分开的常数相乘。它是一个常数 — 圆周率 — 在一个平方根下。我们只是用了错误的符号写它。

## 傅里叶变换

```math
\hat{f}(\xi) = \int_{-\infty}^{\infty} f(x)\,e^{-i\tau\xi x}\,dx
```

自然频率变量是配 $\tau$ 的,不是 $2\pi$。每本信号处理教科书都在和 $2\pi$ 该放哪里搏斗 — 正变换?逆变换?分成 $\sqrt{2\pi}$?这些都是用错常数的症状。用 $\tau$,约定毫不含糊。

> :angrygoose: 傅里叶约定之战浪费的集体时间比应用数学里任何其他记号纠纷都多。$2\pi$ 该放哪里至少有四种常见约定。用 $\tau$,只有一种:放在指数里。完事。每一种约定都在试图说"用 $\tau$",同时被迫写"$2\pi$"。

## 柯西积分公式

```math
f(a) = \frac{1}{i\tau}\oint_\gamma \frac{f(z)}{z - a}\,dz
```

留数定理:$\oint f(z)\,dz = i\tau \sum \text{Res}$。当你在复平面里绕一个轮廓时,在复分析中出现的因子是 $\tau$ — 复平面里的一整圈。到处写 $2\pi i$ 只是用额外步骤写 $i\tau$。

## 物理学

这个模式在物理学里到处都是:

| 公式 | 用 $\tau$ | 用 $\pi$ |
|---|---|---|
| 角频率 | $\omega = \tau f$ | $\omega = 2\pi f$ |
| 约化普朗克常数 | $\hbar = h/\tau$ | $\hbar = h/(2\pi)$ |
| 库仑定律 | $\frac{1}{\tau\varepsilon_0}\frac{2q}{r}$(单位长度) | $\frac{1}{2\pi\varepsilon_0}\frac{q}{r}$ |
| 磁导率 | $\mu_0 = \tau \times 2 \times 10^{-7}$ H/m | $\mu_0 = 4\pi \times 10^{-7}$ H/m |

> :happygoose: $\hbar$ 存在是因为 $h$ 是用 $\pi$ 而不是 $\tau$ 定义的。如果我们一开始就用 $\tau$,$\hbar$ 就只会是 $h$,每本量子力学教科书都会稍微短一些。我们发明了一整个符号 — $\hbar$ — 作为选错圆周率的补丁。让这件事沉淀一下。

## 转换成本为零

在你自己的工作里从 $\pi$ 换到 $\tau$ 不花什么:

1. 在笔记本或代码库的顶部定义 $\tau = 2\pi$
2. 把所有 $2\pi$ 替换为 $\tau$
3. 把裸 $\pi$(半圈)替换为 $\tau/2$

Python:`from math import tau` — 自 3.6 起就在标准库里。
Rust:`std::f64::consts::TAU` — 自 1.47 起可用。
C++:`std::numbers::pi * 2` 或定义自己的常数。
JavaScript:`Math.PI * 2` 或定义 `const TAU = 2 * Math.PI`。

语言已经知道了。教科书在追赶。

> :sarcasticgoose: "可所有教科书都用 $\pi$!" 所有教科书以前也都说地球是宇宙的中心。流行不等于正确。正确的常数是让每个公式更简单的那个,而那个常数是 $\tau$。你的编译器已经支持了。你的记号也应该。

## 为什么这重要

这不是关于审美。这是关于认知负荷。

每次学生写 $2\pi$ 并不得不记得它意味着"一整圈",那就是一个单位的注意力花在约定上而不是数学上。每次物理学家写 $\hbar$ 而不是 $h$ 并解释"它是 $h$ 除以 $2\pi$",那就是一句话花在历史包袱上而不是物理上。

Tau 不让数学变简单。它让记号忠于几何。四分之一圈看起来就是四分之一。完整周期看起来就是一个常数。傅里叶分析和概率里的归一化因子都干净。圆面积里的 $1/2$ 因子揭示了微积分,而不是隐藏它。

我们 300 年前选错了常数。我们从那时起就一直用因子 2 打补丁。这个补丁就是 $\tau$。
