# 物理基础

从计算的视角看物理 — 公式作为可复用的构建块,守恒定律作为不变量,量纲分析作为调试工具。

## 涵盖内容

### 即将推出

- **运动学与动力学**:离散化的 F=ma、抛体运动、谐振子
- **能量与动量**:守恒作为不变量,用能量方法构建问题
- **量纲分析**:物理学家的类型检查器 — 在计算前抓住错误
- **计算物理**:时间步进、数值稳定性、Euler vs Runge-Kutta
- **统计力学类比**:模拟退火、随机游走、扩散作为搜索
- **优化地形**:势能面、梯度流、鞍点

## 方法

- **先公式,后直觉**:陈述结果,然后解释为什么是对的。
- **计算化框架**:每个物理概念都连接到一个数值方法或算法。
- **单位作为类型**:量纲分析抓 bug,就像类型系统在代码中抓 bug 一样。
- **鹅角色** 就近似进行辩论,让你知道何时该信任它们。
